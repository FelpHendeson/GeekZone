O que é HTML?


HTML, sigla para "Hypertext Markup Language" (Linguagem de Marcação de Hipertexto, em português), é uma linguagem de marcação usada para criar páginas da web. Ela é a espinha dorsal da maioria dos sites na internet. HTML permite que os desenvolvedores da web estruturem o conteúdo de uma página da web, definindo elementos como títulos, parágrafos, listas, links, imagens e muitos outros elementos.




HTML utiliza "tags" (etiquetas) para definir elementos e sua estrutura na página. Cada tag é cercada por colchetes angulares ("<" e ">"). Por exemplo, para criar um título, você usaria uma tag h1 para indicar o início do título e h1 para indicar o final do título.
